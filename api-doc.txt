

## 🔑 **Auth & Users**

* Register → `POST https://dinerestro.onrender.com/users/register/`
* Login → `POST https://dinerestro.onrender.com/users/login/`
* Refresh JWT → `POST https://dinerestro.onrender.com/users/token/refresh/`

---

## 🍴 **Foods & Categories**

### Categories

* List categories (user) → `GET https://dinerestro.onrender.com/foods/categories/`
* Category detail (user) → `GET https://dinerestro.onrender.com/foods/categories/<id>/`
* Admin list/create → `GET/POST https://dinerestro.onrender.com/foods/categories/admin/`
* Admin detail → `GET/PUT/PATCH/DELETE https://dinerestro.onrender.com/foods/categories/admin/<id>/`

### Foods

* List foods (user) → `GET https://dinerestro.onrender.com/foods/foods/`
* Food detail (user) → `GET https://dinerestro.onrender.com/foods/foods/<id>/`
* Admin list/create → `GET/POST https://dinerestro.onrender.com/foods/foods/admin/`
* Admin detail → `GET/PUT/PATCH/DELETE https://dinerestro.onrender.com/foods/foods/admin/<id>/`

---

## 🍽 **Table Booking**

### Tables

* List tables (user) → `GET https://dinerestro.onrender.com/table-booking/tables/`
* Admin list/create → `GET/POST https://dinerestro.onrender.com/table-booking/tables/admin/`

### Bookings

* List user bookings → `GET https://dinerestro.onrender.com/table-booking/bookings/`
* Create booking → `POST https://dinerestro.onrender.com/table-booking/bookings/create/`
* Admin list/create → `GET/POST https://dinerestro.onrender.com/table-booking/bookings/admin/`

---

## 🎉 **Events**

* List events (user) → `GET https://dinerestro.onrender.com/events/list/`
* Event detail (user) → `GET https://dinerestro.onrender.com/events/details/<id>/`
* Create event (admin) → `POST https://dinerestro.onrender.com/events/create/`
* Update event (admin) → `PUT/PATCH https://dinerestro.onrender.com/events/update/<id>/`
* Delete event (admin) → `DELETE https://dinerestro.onrender.com/events/delete/<id>/`
